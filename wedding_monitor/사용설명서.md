# 예식장 예약 모니터링 프로그램 사용 설명서

## 📌 이 프로그램은 무엇인가요?

**예식장 예약 자동 확인 프로그램**입니다.

원하는 날짜에 예식장이 꽉 차 있어도, 혹시 취소가 나면 자동으로 확인해서 **휴대폰으로 알림**을 보내줍니다.

---

## 🎯 프로그램이 하는 일

1. ⏰ **1시간마다 자동으로 예식장 홈페이지 확인**
2. 📱 **예약 가능하면 텔레그램으로 알림**
3. ✅ **원하시면 자동으로 예약 신청까지!**

---

## 🏛️ 지원하는 예식장

### 서울대 연구공원 웨딩홀
- 알림 + 자동 예약 가능
- 시간대: 오전 11시, 오후 1시, 3시, 5시, 6시30분

### 서울대 이라운지
- 알림만 가능
- 시간대: 11:00, 14:00, 17:00

---

## 🚀 사용 방법

### 1단계: 프로그램 실행

`자동예약모니터링.exe` 파일을 **더블클릭**

> 💡 설치 필요 없습니다! 바로 실행됩니다.

---

### 2단계: 텔레그램 봇 만들기

알림을 받으려면 텔레그램 봇을 만들어야 합니다.

#### 📱 텔레그램 봇 만드는 법

1. **텔레그램 앱** 실행
2. 검색창에 **`@BotFather`** 입력
3. `/newbot` 입력
4. 봇 이름 입력 (예: `우리결혼알림`)
5. 봇 사용자명 입력 (예: `wedding_alarm_bot`)
   - ⚠️ 반드시 `_bot`으로 끝나야 합니다
6. **Bot Token 복사** (예: `123456789:ABCdef...`)
   - 메모장에 저장해두세요!

#### 🆔 Chat ID 얻는 법

1. 텔레그램에서 **`@userinfobot`** 검색
2. `/start` 입력
3. **숫자(Chat ID) 복사** (예: `987654321`)
   - 메모장에 저장해두세요!

#### ✅ 봇과 대화 시작

1. 본인이 만든 봇 찾기 (예: `우리결혼알림`)
2. `/start` 입력
   - ⚠️ **이 단계를 꼭 해야 알림이 옵니다!**

#### 🔁 두 번째 봇 (선택)

- 2명에게 알림 보내려면 위 과정 반복
- 봇 이름만 다르게 (예: `우리결혼알림2`)

---

### 3단계: 프로그램 설정

프로그램을 실행하면 이런 화면이 나옵니다:

#### ① 날짜 설정

**방법 1: 기간으로 선택**
- "기간 설정 사용" ☑️
- 시작일: 2025-11-01
- 종료일: 2026-06-30
- 요일: "토요일만" / "일요일만" / "토일 둘 다"

**방법 2: 특정 날짜만 선택**
- "특정 날짜 사용" ☑️
- 달력에서 날짜 선택 → "추가" 클릭
- 여러 개 추가 가능

> 💡 두 방법 동시 사용도 가능합니다!

#### ② 시간대 설정

**서울대 연구공원 웨딩홀**
- 각 시간대마다 설정:
  - ☑️ 체크 = 확인함
  - **"알림+자동예약"** = 예약 가능하면 자동 신청
  - **"알림만"** = 알림만 받고 직접 예약

**서울대 이라운지**
- ☑️ 체크만 가능 (알림만)

#### ③ 텔레그램 설정

**텔레그램 1 (필수)**
- Bot Token: 위에서 복사한 Bot Token 붙여넣기
- Chat ID: 위에서 복사한 Chat ID 붙여넣기

**텔레그램 2 (선택)**
- 2명이 알림 받으려면 두 번째 봇 정보 입력
- 안 쓰려면 비워두기

---

### 4단계: 시작!

1. **"설정 저장"** 클릭
2. **"시작하기"** 클릭
3. 텔레그램으로 "프로그램 시작" 알림이 오면 성공!

---

## 📱 알림 받으면 어떻게 하나요?

### 알림 예시

```
🔔 예약 가능 발견!

📍 서울대 연구공원 웨딩홀
📅 2026년 05월 24일 (토)
⏰ 오전 11시

상태 변경: 예약완료 → 예약가능
```

### 자동 예약된 경우

```
✅ 예약 신청 완료!

📍 서울대 연구공원 웨딩홀
📅 2026년 05월 24일 (토)
⏰ 오전 11시

신청 정보:
👰‍♂️ 신랑: 황태준 (010-3096-0329)
👰‍♀️ 신부: 박여경 (010-3895-8291)

⚠️ 중요!
예식장에서 전화 연락이 올 예정입니다.
계약금 입금 전까지는 예약이 확정되지 않습니다.
```

### 알림 받으면 해야 할 일

**자동 예약된 경우:**
1. 예식장 전화 대기
2. 전화 오면 예약 확인
3. 계약금 입금하면 예약 확정!

**알림만 받은 경우:**
1. 바로 예식장 홈페이지 접속
2. 직접 예약 신청
3. 또는 전화로 문의

## ⚠️ 중요한 주의사항

### ✅ 꼭 지켜주세요

1. **PC를 계속 켜놓아야 합니다**
   - 프로그램이 꺼지면 확인을 안 합니다
   - 절전 모드도 해제하세요
   - 모니터만 끄는 건 괜찮습니다

2. **인터넷 연결 유지**
   - 인터넷이 끊기면 확인이 안 됩니다

3. **자동 예약은 "신청"만 합니다**
   - 예식장 직원 확인 필요
   - 계약금 입금해야 최종 확정
   - 다른 사람이 먼저 계약금 넣으면 취소될 수 있음

4. **Bot Token 유출 주의**
   - 절대 다른 사람에게 알려주지 마세요

## 🔧 문제가 생겼을 때

### 프로그램이 "인증 실패" 뜨면서 안 열려요

**이유**: 이 프로그램은 특정 컴퓨터에서만 실행됩니다.

**해결**:
- 처음 받은 컴퓨터에서만 실행하세요
- 다른 컴퓨터로 옮기면 안 됩니다

### 텔레그램 알림이 안 와요

**확인사항**:
1. Bot Token 정확한가요?
2. Chat ID 정확한가요?
3. 봇과 `/start` 했나요?
4. 인터넷 연결되어 있나요?

**테스트**:
- "시작하기" 누르면 "프로그램 시작" 알림이 와야 정상

### Windows Defender가 차단해요

**이유**: exe 파일이라서 보안 경고가 뜰 수 있습니다.

**해결**:
1. "추가 정보" 클릭
2. "실행" 클릭
3. 또는 프로그램을 예외로 추가

## ❓ 자주 묻는 질문

### Q. PC를 끄면 안 되나요?
**A.** 네, PC를 끄면 프로그램이 멈춥니다.
- 항상 켜놓는 PC에서 실행하세요
- 절전 모드 해제 필수

### Q. 핸드폰으로는 안 되나요?
**A.** PC 전용 프로그램입니다.

### Q. 알림은 누구한테 가나요?
**A.** Chat ID 입력한 사람에게만 갑니다.
- 2명한테 보내려면 텔레그램 2개 설정

### Q. 자동 예약하면 무조건 예약되나요?
**A.** 아닙니다!
- 예약 "신청"만 자동
- 직원 확인 + 계약금 입금 필요
- 다른 사람이 먼저 계약금 넣으면 취소됨

### Q. 여러 날짜 확인할 수 있나요?
**A.** 네!
- 기간으로 여러 달 확인 가능
- 특정 날짜만 선택도 가능
- 두 방법 동시 사용도 가능

### Q. 확인 주기를 더 짧게 할 수 있나요?
**A.** 1시간으로 고정되어 있습니다.
- 너무 자주 확인하면 예식장 홈페이지에 부담
- 1시간이면 충분히 빠릅니다

### Q. 예약자 이름을 바꿀 수 있나요?
**A.** 이 프로그램은 **황태준, 박여경** 정보로 고정되어 있습니다.
- exe 파일이라 수정 불가능
- 다른 정보가 필요하면 개발자에게 문의

## 📊 사용 예시

### 예시 1: 2026년 5월 토요일, 오전 11시만 자동 예약

1. **날짜 설정**
   - "기간 설정 사용" ☑️
   - 시작일: 2026-05-01
   - 종료일: 2026-05-31
   - 요일: "토요일만"

2. **시간대 설정**
   - 연구공원 "오전 11시" ☑️ + "알림+자동예약"
   - 나머지 시간대는 체크 해제

3. **텔레그램 설정** → **설정 저장** → **시작하기**

### 예시 2: 특정 날짜 5개, 모든 시간대, 알림만

1. **날짜 설정**
   - "특정 날짜 사용" ☑️
   - 2026-05-02, 05-09, 05-16, 05-23, 05-24 추가

2. **시간대 설정**
   - 연구공원 모든 시간대 ☑️ + "알림만"
   - 이라운지 모든 시간대 ☑️

3. **텔레그램 설정** → **설정 저장** → **시작하기**
