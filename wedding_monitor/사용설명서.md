# 예식장 예약 모니터링 프로그램 - 사용 설명서

## 📌 이 프로그램은 무엇인가요?

결혼 준비하시는 분들을 위한 **예식장 예약 자동 확인 프로그램**입니다.

### 이런 분들에게 필요합니다
- ✅ 원하는 날짜에 예식장이 꽉 차서 예약 못하신 분
- ✅ 매일 예식장 홈페이지 들어가서 확인하기 귀찮으신 분
- ✅ 혹시 취소되면 바로 알림 받고 싶으신 분

### 프로그램이 하는 일
1. **자동으로 확인**: 1시간마다 예식장 홈페이지를 자동으로 확인합니다
2. **자리 나면 알림**: 예약 가능한 날짜가 생기면 휴대폰으로 알림을 보내줍니다
3. **자동 예약**: 원하시면 자동으로 예약 신청까지 해줍니다

---

## 🎯 지원하는 예식장

### 1. 서울대 연구공원 웨딩홀
- **알림** + **자동 예약** 둘 다 가능
- 시간대: 오전 11시, 오후 1시, 3시, 5시, 6시30분

### 2. 서울대 이라운지
- **알림만** 가능 (자동 예약 불가)
- 시간대: 11:00, 14:00, 17:00

## 💻 프로그램 설치하기

### 준비물
1. Windows 컴퓨터 (항상 켜놓을 수 있는 PC)
2. 인터넷 연결
3. 휴대폰 (텔레그램 앱 설치)

### 설치 순서

#### 1단계: Python 설치 (5분)
1. https://www.python.org/downloads/ 접속
2. 노란색 "Download Python" 버튼 클릭
3. 다운로드된 파일 실행
4. ⚠️ **중요**: 맨 아래 "Add Python to PATH" 체크 필수!
5. "Install Now" 클릭

#### 2단계: 프로그램 파일 다운로드
1. 이 폴더(`wedding_monitor`)를 바탕화면에 복사

#### 3단계: 필요한 기능 설치 (5분)
1. `wedding_monitor` 폴더에서 마우스 우클릭
2. "여기서 터미널 열기" 또는 "PowerShell 여기서 열기" 클릭
3. 아래 명령어 **복사해서 붙여넣기** 후 Enter:
   ```
   pip install -r requirements.txt
   ```
4. 설치가 끝날 때까지 대기 (2~3분)

## 📱 텔레그램 봇 만들기 (10분)

알림을 받으려면 텔레그램 봇을 만들어야 합니다. 2개까지 만들 수 있습니다!

### 텔레그램 봇 1개 만들기

#### 1단계: 봇 만들기
1. 휴대폰에서 텔레그램 앱 실행
2. 검색창에 `@BotFather` 입력해서 찾기
3. 대화 시작하고 `/newbot` 입력
4. 봇 이름 입력 (예: `우리결혼예약알림`)
5. 봇 사용자명 입력 (예: `wedding_alarm_bot`)
   - ⚠️ 반드시 `_bot`으로 끝나야 합니다
6. **Bot Token이 나옵니다** (예: `123456789:ABCdefGHI...`)
   - 이걸 어딘가에 복사해두세요! 나중에 필요합니다

#### 2단계: Chat ID 얻기
1. 텔레그램 검색창에 `@userinfobot` 입력
2. 대화 시작하고 `/start` 입력
3. **숫자(Chat ID)가 나옵니다** (예: `987654321`)
   - 이것도 복사해두세요!

#### 3단계: 봇과 대화 시작
1. 텔레그램에서 본인이 만든 봇 찾기 (예: `우리결혼예약알림`)
2. 대화 시작하고 `/start` 입력
   - ⚠️ 이 단계를 안 하면 알림이 안 옵니다!

### 두 번째 봇 만들기 (선택사항)
- 2명이 동시에 알림 받고 싶으면 위 과정을 한 번 더 반복
- 봇 이름만 다르게 (예: `우리결혼예약알림2`)

## 🚀 프로그램 사용하기

### 프로그램 실행
1. `wedding_monitor` 폴더에서 `main.py` 더블클릭
2. 프로그램 창이 뜹니다

### 화면 설명

프로그램 창은 크게 4개 영역으로 나뉩니다:

#### ① 날짜 설정
**방법 1: 기간으로 설정**
- "기간 설정 사용" 체크
- 시작일: 확인 시작할 날짜 (예: 2025년 11월 1일)
- 종료일: 확인 끝날 날짜 (예: 2026년 6월 30일)
- 요일: 토요일만 / 일요일만 / 토일 둘 다

**방법 2: 특정 날짜만 선택**
- "특정 날짜 사용" 체크
- 달력에서 원하는 날짜 클릭
- "추가" 버튼 클릭
- 여러 날짜 추가 가능 (예: 11월 29일, 3월 4일 등)

#### ② 시간대 설정

**서울대 연구공원 웨딩홀**
- 각 시간대마다 선택:
  - ☑️ 체크 = 이 시간대 확인함
  - "알림+자동예약" = 예약 가능하면 자동으로 예약 신청
  - "알림만" = 예약 가능하면 알림만 보냄 (직접 예약해야 함)

**서울대 이라운지**
- ☑️ 체크만 가능 (알림만 받을 수 있음)

#### ③ 텔레그램 설정

**텔레그램 1 (필수)**
- Bot Token: 위에서 복사한 Bot Token 붙여넣기
- Chat ID: 위에서 복사한 Chat ID 붙여넣기

**텔레그램 2 (선택)**
- 2명이 알림 받고 싶으면 두 번째 봇 정보 입력
- 안 쓰려면 비워두면 됨

#### ④ 버튼

1. **설정 저장**: 지금까지 입력한 정보 저장
2. **시작하기**: 모니터링 시작! (1시간마다 자동 확인)
3. **중지**: 모니터링 멈춤
4. **종료**: 프로그램 닫기

### 예시 1: 2026년 3월 토요일 오전 11시 자동 예약 받기

   - "기간 설정 사용" ☑️
   - 시작일: 2026-03-01
   - 종료일: 2026-03-31
   - 요일: "토요일만" 선택

2. **시간대 설정**
   - 연구공원: "오전 11시" ☑️ + "알림+자동예약" 선택

3. **텔레그램 설정**
   - Bot Token, Chat ID 입력

4. **설정 저장** → **시작하기** 클릭

### 예시 2: 특정 날짜 3개만 확인 (알림만)

1. **날짜 설정**
   - "특정 날짜 사용" ☑️
   - 달력에서 2026-11-29 선택 → 추가
   - 달력에서 2026-03-04 선택 → 추가
   - 달력에서 2026-06-08 선택 → 추가

2. **시간대 설정**
   - 연구공원: "오전 11시" ☑️ + "알림만" 선택
   - 연구공원: "오후 1시" ☑️ + "알림만" 선택

3. **텔레그램 설정**
   - Bot Token, Chat ID 입력

4. **설정 저장** → **시작하기** 클릭

## 🔧 문제 해결

### 프로그램이 실행 안 돼요
**증상**: main.py 더블클릭해도 창이 안 뜸

**해결**:
1. Python 설치 확인
2. 명령 프롬프트(CMD)에서 실행:
   ```
   cd wedding_monitor
   python main.py
   ```
3. 오류 메시지 확인

### "ModuleNotFoundError" 오류가 나요
**증상**: `No module named 'psutil'` 같은 오류

**해결**:
```
pip install -r requirements.txt
```

### 텔레그램 알림이 안 와요
**확인사항**:
1. ✅ Bot Token이 정확한가요?
2. ✅ Chat ID가 정확한가요?
3. ✅ 봇과 대화 시작(`/start`)을 했나요?
4. ✅ 인터넷 연결이 되어 있나요?

**테스트 방법**:
- "시작하기" 누르면 "프로그램 시작되었습니다" 알림이 와야 정상

### Chrome 관련 오류가 나요
**증상**: `ChromeDriver` 오류

**해결**:
1. Chrome 브라우저 설치 확인
2. Chrome 최신 버전으로 업데이트

### 예약이 자동으로 안 돼요
**확인사항**:
1. ✅ "알림+자동예약"을 선택했나요?
2. ✅ `auto_reservation.py`에 본인 정보가 입력되어 있나요?
3. ✅ 예식장 홈페이지가 정상 작동하나요?

### 프로그램이 "인증 실패" 뜨면서 안 열려요
**증상**: "이 프로그램은 현재 컴퓨터에서 실행할 수 없습니다"

**해결**:
- 이 프로그램은 특정 컴퓨터에서만 실행되도록 설정되어 있습니다
- 다른 컴퓨터에서 실행하려면 개발자에게 문의하세요

## ❓ 자주 묻는 질문

### Q1. PC를 끄면 안 되나요?
**A.** 네, PC를 끄면 프로그램이 멈춥니다.
- 집에서 항상 켜놓는 PC 사용 권장
- 또는 절전 모드만 해제하고 모니터만 꺼도 됩니다

### Q2. 핸드폰으로는 안 되나요?
**A.** 이 프로그램은 Windows PC 전용입니다.
- 핸드폰으로는 실행 불가능합니다

### Q3. 알림은 누구한테 가나요?
**A.** Bot Token과 Chat ID를 입력한 사람에게만 갑니다.
- 2명에게 보내려면 텔레그램 봇 2개 만들기

### Q4. 자동 예약하면 무조건 예약되나요?
**A.** 아닙니다!
- 예약 "신청"만 자동으로 됩니다
- 직원 확인 + 계약금 입금해야 최종 확정
- 다른 사람이 먼저 계약금 넣으면 취소될 수 있음

### Q5. 여러 날짜를 동시에 확인할 수 있나요?
**A.** 네!
- 기간 설정으로 여러 달 확인 가능
- 특정 날짜로 원하는 날만 선택 가능
- 두 방법 동시 사용도 가능

### Q6. 확인 주기를 더 짧게 할 수 있나요?
**A.** 현재 1시간으로 고정되어 있습니다.
- 너무 자주 확인하면 예식장 홈페이지에 부담
- 1시간이면 충분히 빠른 편입니다

### Q7. exe 파일로 실행하고 싶어요
**A.** 가능합니다!
```
pyinstaller --onefile --noconsole --name="자동예약모니터링" --icon=icon.ico --hidden-import=psutil --hidden-import=psutil._pswindows --hidden-import=psutil._psutil_windows --hidden-import=psutil._common main.py
```
- 실행 후 `dist` 폴더에 exe 파일 생성
- 더블클릭으로 바로 실행 가능
